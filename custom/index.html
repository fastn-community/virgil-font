FTD virgil-font package

Package Name
fastn-community.github.io/virgil-font
This FTD font virgil-font package is for your ftd web projects. To Use this virgil-font inside FTD project follow below instructions.

How to use virgil-font?

Add this to your FASTN.ftd
-- fastn.dependency: fastn-community.github.io/virgil-font

After adding virgil-font dependency inside your FASTN.ftd file, You need to add virgil-font assets and import it inside your .ftd files where you are using this font for your custom font types.

To include virgil-font into your ftd file follow below instructions.
-- import: fastn-community.github.io/virgil-font/assets as virgil-font-assets

How to create a custom virgil-font type?

We don't recommend creating custom types. We recommend to use our virgil-font-typography

If you're using virgil-font-typography and still your designer has introduced new font size which are not included or part of our virgil-font-typography then only define custom types.

We hope you shouldn't need to define a custom type.

Below is an example of creating a custom ftd.type using virgil-font.
Input
-- ftd.type virgil-font-size-12-desktop:
size.px: 12
weight: 400
font-family: $virgil-font-assets.fonts.virgil


-- ftd.type virgil-font-size-12-mobile:
size.px: 12
weight: 400
font-family: $virgil-font-assets.fonts.virgil

-- ftd.responsive-type virgil-font-size-12-400:
desktop: $virgil-font-size-12-desktop
mobile: $virgil-font-size-12-mobile

-- ftd.text: 
role: $virgil-font-size-12-400
color: $inherited.colors.text-strong
